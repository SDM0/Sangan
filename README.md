# Sangan

Adds a Structure Pack for [nh6574](https://github.com/nh6574)'s modern Yu-Gi-Oh! mod [JoyousSpring](https://github.com/nh6574/JoyousSpring).

This add-on was made for specific archetype to be fetched more easily.

The booster spawn type can be configured in the in-game mod config.

## Installation
- Requires [Steamodded](https://github.com/Steamopollys/Steamodded/) 1814a, [Lovely](https://github.com/ethangreen-dev/lovely-injector) 0.9.0 and [JoyousSpring](https://github.com/nh6574/JoyousSpring) 1.0.0
- Download `Sangan.zip` through the green "Code" button, then "Download ZIP"
- Unzip it into `AppData\Roaming\Balatro\Mods`
